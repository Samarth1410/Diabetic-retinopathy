# Diabetic Retinopathy Detection

Fine-tuned VGG16 to classify diabetic retinopathy severity into 5 grades
from retinal fundus images using the APTOS 2019 dataset.

## Project Structure
- `01_eda.ipynb` — Exploratory data analysis
- `02_preprocess.ipynb` — Augmentation & DataLoaders
- `03_train.ipynb` — Transfer learning with VGG16
- `04_evaluate.ipynb` — Metrics, confusion matrix, GradCAM
- `05_inference.ipynb` — Single & batch prediction

## Tech Stack
Python, PyTorch, VGG16, Transfer Learning, Computer Vision

## Dataset
APTOS 2019 Blindness Detection (Kaggle)
